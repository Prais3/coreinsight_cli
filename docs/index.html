CoreInsight

A local-first, hardware-aware AI performance profiler and optimizer.

pip install coreinsight-cli

More than a code generator

CoreInsight parses Python, C++, and CUDA code to identify hardware bottlenecks like CPU cache thrashing and CUDA warp divergence. It generates optimized versions using local or cloud LLMs, but does not stop there.

Every optimization runs through a two-pillar mathematical verification system inside isolated Docker sandboxes:

  • Speedup Integrity: Recomputes speedup from raw timing columns and cross-checks against reported values to strictly flag LLM hallucinations.
  • Output Correctness: Runs both original and optimized functions on identical, dynamically generated test inputs to ensure outputs match within a float tolerance.
